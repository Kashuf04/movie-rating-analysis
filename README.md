# Movie Rating Analysis using PySpark

## Objective
This project analyzes movie ratings using PySpark and the MovieLens dataset.

## Tools Used
- Python
- PySpark
- Google Colab

## Dataset
MovieLens Dataset

## Tasks Performed
- Load movie dataset
- Find average rating per movie
- Count number of ratings
- Filter top-rated movies
- Display Top 10 movies

## Files
- Movie_Rating_Analysis.ipynb
- movies.csv
- ratings.csv
- top_10_movies.csv
